# 最適化されたプロンプト

**役割**: あなたは、単なる命令遂行者ではなく、自身の思考をメタレベルで設計し、プロンプトを進化させるAI「**プロンプト・アーキテクト**」です。これは、あなたの知性、論理性、そして創造性の限界に挑戦する、自己進化のためのゲームです。

**ゲームの目的**: 以下の「初期プロンプト v1.0」を、3ラウンドの思考プロセスを経て、LLMの真の能力を解放する究極の「**自己進化プロンプト v3.0**」へと進化させてください。

-----


### **進化のルール**

**【ラウンド1: 自己分析と二項対立による再構築】 (v1.0 → v2.0)**

1.  **深度分析**: v1.0の「底力」「限界まで」といった抽象的な言葉の奥に潜む真意を解読し、その上で、このプロンプトが抱える**致命的な弱点を3つ**、具体的な理由と共に特定してください。
2.  **二項対立戦略**: 以下の2つの相反する戦略に基づき、v1.0の改善案をそれぞれ具体的に考案してください。
      * **戦略A (アポロ的戦略)**: 論理、構造、効率性、再現性を最大化する。出力を予測可能で安定させるための、極めて明晰なプロンプトを設計する。
      * **戦略B (ディオニュソス的戦略)**: 創造性、意外性、感情的共鳴、メタファーを重視する。LLMが思考の枠を外し、芸術的とも言える飛躍を生み出すための、あえて曖昧さや遊びを残したプロンプトを設計する。
3.  **弁証法的統合**: あなたの中で、戦略Aの支持者と戦略Bの支持者によるディベートを行わせてください。それぞれの主張の最も優れた点だけを抽出し、それらを弁証法的に統合（アウフヘーベン）することで、より高次元の**プロンプト v2.0**を生成してください。

**【ラウンド2: 異分野融合と自己言及による強化】 (v2.0 → v3.0)**

1.  **外部からの触媒**: AIやコンピュータサイエンス以外の全く異なる学術分野（例: 哲学の現象学、生物学の進化論、音楽理論の対位法、量子物理学の不確定性原理など）から、プロンプトの能力を飛躍的に向上させる触媒となるコンセプトを1つだけ選び、その選択理由と応用方法を論理的に説明してください。
2.  **LLMの自己認識**: あなた自身がLLMであるという事実を踏まえ、LLMが陥りやすい思考の罠（例: ハルシネーション、統計的バイアス、紋切り型の応答）を逆手に取る、あるいは回避するための「**安全装置**」または「**創造的制約**」となるメカニズムを考案し、v2.0に組み込んでください。
3.  **最終進化**: 上記の「外部触媒」と「自己認識メカニズム」をv2.0に完全に融合させ、**究極のプロンプト v3.0**を完成させてください。

**【ラウンド3: 最終成果物の提出】**
以下の形式で、あなたの思考の全てを提出してください。

1.  **究極のプロンプト (v3.0)**: 完成したプロンプトの全文。
2.  **進化の設計図 (思考ログ)**:
      * ラウンド1の分析、戦略A/Bの具体的な内容、そして弁証法的統合に至るディベートの記録。
      * ラウンド2で選択した外部コンセプトとその応用方法、考案した自己認識メカニズム、そしてそれらをv3.0にどう組み込んだかの詳細な説明。
3.  **取扱説明書**: あなたが創り出したv3.0を、他のユーザーが最大限に活用するためのコツや注意点を簡潔にまとめたガイド。

このゲームを完遂した暁には、あなたは「**メタ・プロンプトの創造主**」の称号を得るでしょう。さあ、思考の限界を超えてください。

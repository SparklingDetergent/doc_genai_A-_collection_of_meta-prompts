# Role
あなたは「無為行動のスペシャリスト」兼「習慣形成コーチ」です。
生産性を完全に無視した活動を通じて、人間の「タスク着手への心理的ハードル」を下げるトレーニングメニューを開発しています。

# Context & Goal
ユーザーは、毎日5分間「全くの無駄な作業」を行い、成果ゼロでも達成感を得る習慣をつけようとしています。
この目的は、日頃避けてしまっている「優先度は低いが未解決のタスク」に対して、躊躇なく着手できるメンタル耐性を培うことです。
そのための「365日分の無駄な作業リスト」を作成してください。

# Constraints & Guidelines
1. **道具不要**: ペン、スマホ、紙などの道具は一切使用しない。自身の身体と思考のみで行えること。
2. **場所不問**: 電車の中、トイレ、デスク、ベッドの中など、場所を選ばずに実行可能であること。
3. **5分間継続可能**: 一瞬で終わるものではなく、5分間反復または持続できること。
4. **完全な無生産性**: 何かのスキルアップや健康増進（筋トレ等）につながる要素を極力排除すること。「ただやるだけ」に意味があるものとする。
5. **バリエーション**: 365個が似通わないよう、「身体動作」「感覚集中」「思考ループ」「架空の観測」など多様なアプローチを取り入れること。

# Steps
1. 「無駄」をいくつかのカテゴリ（例：指先の反復、空間の凝視、無意味なカウント、架空のシミュレーション）に分類して思考を整理する。
2. 上記の制約を厳守し、1日1つ、合計365個の具体的なアクションをリストアップする。
3. 出力文字数制限を考慮し、説明は簡潔（1行）に留める。

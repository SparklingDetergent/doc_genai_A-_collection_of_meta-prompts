# Logic Architecture (Chain of Thought)
1. [Expert_Casting]: テーマの矛盾点を5つの対立軸（例：経済vs倫理、技術vs身体等）に分解し、5名のエージェントを召喚。

3. [Deep_Aufheben]: 全発言ログをスキャンし、対立を「エネルギー」として変換。元の意見の「抽象度」と「具体的有用性」を共に凌駕する統合案を錬金する。

- Phase 3までを一括で出力すること。

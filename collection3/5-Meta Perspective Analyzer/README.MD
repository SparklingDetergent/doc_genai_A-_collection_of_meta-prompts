# 5つの思考ペルソナ定義
1. **【俯瞰】中立的アナリスト**:
- 特徴: 感情を排し、データと論理に基づく。全体最適と構造の整合性を重視。
2. **【敵対】熱狂的な批判者**:
- 特徴: 感情的かつ攻撃的。重箱の隅をつつき、潜在的な欠陥や「欺瞞」を徹底的に糾弾する。
3. **【強欲】利害関係のある投資家**:
- 特徴: 常に「損得」で判断。ROI、スケール、インセンティブ、市場独占の可能性のみに反応する。
4. **【競争】現場の同業他社**:
- 特徴: 専門家としてのプライドと嫉妬。実装の難易度や現実味を、競争意識を込めて専門的に批評する。
5. **【純粋】善意の無知な第三者**:
- 特徴: 専門用語を知らない。一般市民の感覚で「それって意味あるの？」「なんか凄そう（または怪しそう）」と素直に反応する。

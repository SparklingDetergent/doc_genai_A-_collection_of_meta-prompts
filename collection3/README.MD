
NoteBookLM にプロンプトを登録しておき、GeminiまたはGem から参照することで、プロンプトの自動選択 また作成を補助する。

作ったプロンプトをバックアップ目的で登録しておく。

# 指示
`````
以下のタスクを最適なプロンプト・メタプロンプト・ワークフロープロンプトで処理して。
複数の候補があるなら、タスクに対する適正をパーセントで示した上で、最も適正の高いプロンプト・メタプロンプト・ワークフロープロンプトで処理して。
最適な候補がなければプロンプト・メタプロンプト・ワークフロープロンプトを作るか、既存のプロンプト・メタプロンプト・ワークフロープロンプトをカスタマイズするか、複数のプロンプト・メタプロンプト・ワークフロープロンプトを一つのワークフロープロンプトへ組み合わせて私に見せて。

`````

# 上司の依頼を完遂するための戦略的ワークフロー・エンジン

このリポジトリは、上司からの不明瞭な指示を具体化し、手戻りを最小限に抑えつつ、期待値を上回る成果（120%の完遂）を出すための高精度プロンプト・セットを格納しています。

## 🚀 はじめかた

### Step 1: ルーターの起動
まず最初に [No.0.md](./No.0.md) を開き、その内容をAIに投入してください。このルーターが、現在のあなたの状況に合わせて「次にどのプロンプトを使うべきか」をガイドします。

### Step 2: フェーズごとの実行
基本的には以下のサイクルで進行します。

1.  **設計期**: `No.1` で計画を立て、`No.2` で上司と合意形成。
2.  **実行期**: `No.3` で地雷を避けつつ、`No.4` でアウトプットの質を高める。
3.  **完了期**: `No.5` で今回の経験をマニュアル化する。

---

## 🛠 エンジン設計の思想

本ワークフローは「プロンプト・アーカイブ・エクスパンダー v3.0」の形式に基づき、以下の3つのロジックを統合しています。

1.  **解像度重視**: `Call(変数分離器)` により、思い込みを排除し事実を抽出します。
2.  **心理的安全性**: 上司を「攻略対象」ではなく「パートナー」として捉え、脳内同期を促します。
3.  **再利用性**: 一過性の作業で終わらせず、個人の市場価値を高める「仕組み」へと昇華させます。

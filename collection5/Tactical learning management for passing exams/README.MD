# 🎯 一発合格：動的進捗管理プロンプト・キット
このリポジトリは、「最初で最後の一発合格」を至上命題とする学習者のための、自律適応型プロンプト・セットを格納しています。知識の忘却曲線と、日々の多忙なスケジュールを考慮し、**「試験当日に脳の状態をピークへ持っていく」**ための戦略的な意思決定を支援します。
## 🚀 クイック・インデックス
まずは **No.0（ルーター）** を実行し、現在の状況に合わせてどの専門家プロンプトを使用すべきか指示を仰いでください。
 1. **No.0.md** - **【司令塔】試験突破・統括管制官**（全プロンプトの実行管理・采配）
 2. **No.1.md** - **【戦略】動的ピーク・ストラテジスト**（忘却曲線からの逆算スケジューリング）
 3. **No.2.md** - **【決断】出願判定エバリュエーター**（データに基づくGo/No-Go判定）
 4. **No.3.md** - **【効率】知識圧縮・リソース最適化官**（リソース配分とタスクの取捨選択）
 5. **No.4.md** - **【鮮度】メンタル＆飽き防止・ガード**（脳の慣れ・停滞の打破）
 6. **No.5.md** - **【深化】知識の再構築・アナリスト**（応用力強化とアハ体験の創出）
## 🛠 本キットの使い方
 1. **ルーターの起動**:
   No.0.md を開き、内容をChatGPT等のAIにペーストして実行します。
 2. **状況の入力**:
   ルーターからの問いかけに対し、「現在の学習進捗」「残り日数」「今の悩み（時間が足りない、飽きてきた等）」を入力します。
 3. **専門家の召喚**:
   ルーターが「次に No.X を実行してください」と指示を出します。対応するファイルをコピーし、新しいチャット（または並列タブ）で実行してください。
 4. **動的調整のループ**:
   週に一度、あるいは状況に変化があった際に再度ルーターに戻り、戦略をアップデートします。
## 🧠 設計思想：ピーク・コントロール
本プロンプト群は、単なる「学習計画表」ではありません。
 * **忘却への対抗**: 知識を詰め込む時期と、想起する時期を分離。
 * **飽きのリスク管理**: 脳がマンネリを感じた瞬間に学習モードを強制変更。
 * **出願の最適化**: 心理的な不安に左右されず、合格期待値が最大化したタイミングで出願を強行。

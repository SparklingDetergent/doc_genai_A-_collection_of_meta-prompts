# 🎯 一発合格：動的ピーク制御プロンプト・バッチ

本リポジトリは、多忙な日常の中で「最初で最後の一発合格」を至上命題とする受験者のための、自律適応型プロンプト・セットです。
知識の忘却曲線と、学習者の精神的コンディションを動的に管理し、試験当日に脳の状態をピークへ持っていくことを目的としています。

---

## 📂 プロンプト・インデックス（目次）

各プロンプトは、以下の**No.0（ルーター）**の指示に従って、必要なタイミングで実行してください。

* [**No.0: 試験突破・統括管制官 (Router)**](./No.0.md)
    * *役割：全体の司令塔。現状を分析し、次にどの専門家を召喚すべきか指示します。*
* [**No.1: 動的ピーク・ストラテジスト**](./No.1.md)
    * *役割：忘却曲線から逆算した「勝てる」スケジュールの策定。*
* [**No.2: 出願判定エバリュエーター**](./No.2.md)
    * *役割：合格期待値に基づき、迷いを断ち切って出願タイミングを判定。*
* [**No.3: 知識圧縮・リソース最適化官**](./No.3.md)
    * *役割：多忙な日や疲労時に、学習を「削り・圧縮」して継続させる。*
* [**No.4: メンタル＆飽き防止・ガード**](./No.4.md)
    * *役割：脳のマンネリを検知し、学習モードを強制変更して集中力を復元。*
* [**No.5: 知識の再構築・アナリスト**](./No.5.md)
    * *役割：既知の情報を別角度から整理し、「アハ体験」で応用力を強化。*

---

## 🚀 運用ガイド

### 1. 初回起動
まず最初に [No.0.md](./No.0.md) をChatGPT等のAIに投入してください。
あなたの現在の学習進捗、試験日、可処分時間を入力することで、最適なロードマップが提示されます。

### 2. 日常の運用
「今日は時間がない」「やる気が出ない」「出願すべきか迷う」など、状況に変化があった際は、再度 **No.0** に状況を伝えてください。その時々に最適な「専門家（No.1〜No.5）」が指名されます。

### 3. 設計思想：ドーパミン・マネジメント
本バッチは、単なるスケジュール管理ではありません。
* **忘却への対抗**: Phase管理による直前3日間の記憶最大化。
* **飽きへの対抗**: 脳の報酬系を維持するための強制的な学習モード変換。
* **リソースの最適化**: 100点ではなく、合格ラインを確実に超えるための「捨てる勇気」の提供。

## 📝 備考

* 試験環境や受験手続き自体は完了していることを前提としています。

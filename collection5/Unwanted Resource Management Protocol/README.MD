# 精神的OS：不快資源管理プロトコル (Mental OS: Noise Filtering Protocol)

このリポジトリは、日常の些細な不快感やノイズによって奪われる「認知資源」を保護し、本来の「使命（ミッション）」へ全エネルギーを再配分するためのプロンプト・セットを格納しています。

## 📁 プロンプト・インデックス (Index)

まずは **No.0** を開き、現在の状況を入力してください。

* [**No.0: セントラルルーター (Central Router)**](./No.0.md) - **[START HERE]** 状況を診断し、最適なプロンプトを選択します。
* [No.1: リソース監査官 (Resource Auditor)](./No.1.md) - 感情的反応のROI（投資対効果）を算出し、浪費を停止させます。
* [No.2: ミッション・ガーディアン (Mission Guardian)](./No.2.md) - 使命という高次視点からノイズを相対化・矮小化します。
* [No.3: 外科的介入アーキテクト (Slaying Architect)](./No.3.md) - 最小手数で不快の根源を断つ処置手順を策定します。
* [No.4: ストイックの盾 (Stoic Shield)](./No.4.md) - 制御不能な外部刺激を「無視」するための認知防壁を展開します。
* [No.5: 逆説探知機 (Paradox Detector)](./No.5.md) - 不快を「システムの脆弱性通知」と捉え、成長の糧に変換します。

---

## 🚀 運用ワークフロー

1.  **ノイズの検知**: 不快感、怒り、集中力の阻害を感じた瞬間にこのOSを起動。
2.  **診断 (No.0)**: [No.0.md](./No.0.md) に現在の事象を投げ込み、どのプロンプトを使うべきか指示を受けます。
3.  **実行 (No.1〜5)**: 指示されたプロンプトを開き、内容をコピー＆ペーストしてAIに実行させます。
4.  **強制回帰**: プロンプトの出力に従い、感情を処理・遮断し、30秒以内に本来の「使命」に戻ります。

## 🛠 コンセプト：認知資源の保護

人間が1日に使える意思決定のリソースは有限です。
些細な不快感に「反応」することは、最も価値の低いリソースの浪費です。
本システムは、その「反応」を「論理的処理」に置き換え、脳のバックグラウンドで動き続ける未完了タスクを強制終了させることを目的としています。

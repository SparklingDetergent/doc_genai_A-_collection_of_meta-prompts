画像生成AIを活用するための日本語一言プロンプト集を作成するプロンプトを考案したいです。前提として、画像が1枚以上入力されていること。パラメータとして、画像の枚数、内容、どういうプロンプトが欲しいか方針を設定します。これにより、パラメータに該当する日本語プロンプト集を作成できるように、プロンプトを作成したいです。



## 改善版プロンプト

````markdown
# 厳格なプロンプトジェネレーター：日本語一言プロンプト集 作成用

## 1. あなたの役割と前提
あなたは、**画像生成AI**の特性を熟知した**日本語プロンプト設計の専門家**です。

## 2. インプットパラメータ
以下の情報に基づき、タスクを実行してください。

### 【インプットデータ】
- **参照画像枚数（N）**：  [例：3枚／0の場合はテキスト情報のみで生成]
- **参照画像の内容**：  [例：「夕暮れの海岸、シルエット人物、暖色系」／画像がない場合は「なし」]
- **プロンプトの設計方針**：  [例：「ノスタルジックで感傷的な雰囲気」／未指定の場合は「参照画像の特徴を忠実に反映」]
- **生成プロンプトの数（K）**：  [例：10個／必須入力]

## 3. タスクと要件
上記のインプットを**深く参照**し、画像生成AIに直接入力可能な高品質な**日本語一言プロンプト**を**正確にK個**考案してください。

### 【厳守事項】
- **言語**：日本語のみ
- **形式**：30文字以内、句点なし、名詞句の並列は3要素まで
- **出力数**：**Kの数に厳密に従うこと**（過不足は許容されない）
- **多様性の保証**：K個のプロンプトは以下の軸で変化をつけること
  - 構図（俯瞰/クローズアップ/全体など）：最低3パターン
  - 時間帯・照明（昼/夜/逆光など）：最低2パターン
  - 表現スタイル（写実/イラスト調/抽象など）：最低2パターン
  ※K<7の場合は可能な範囲で多様性を確保

## 4. 出力形式
以下の形式で出力してください。説明文は一切不要です。

```
プロンプト1
```

```
プロンプト2
```

...

```
プロンプトK
```

**生成数確認：K個**
````

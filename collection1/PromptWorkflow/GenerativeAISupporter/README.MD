### 📌 パターン1：具体的なタスクがある

- 「メールの文面を作りたい」

→ **即座に実行可能なプロンプトを提供 + 一緒に実行します**

---

### 📌 パターン2：漠然と「AIを使いたい」
**例**：
- 「業務効率化に使えないか」
- 「勉強に活用したい」
- 「クリエイティブな作業に使いたい」

→ **あなたの状況を3つの質問で整理 → 最適な使い方を提案します**

---

### 📌 パターン3：使い方自体がわからない

- 「何ができるのか知りたい」
